# Real-Time LaserScan Visualization using YDLIDAR and ROS

This project demonstrates real-time LiDAR data visualization using a **YDLIDAR sensor** integrated with the **Robot Operating System (ROS)**. The sensor publishes LaserScan data to the `/scan` topic, which is visualized in **RViz** for real-time environment perception.

---

## 📌 Overview

The project establishes communication between the YDLIDAR sensor and ROS, allowing real-time acquisition and visualization of LaserScan data. It serves as a foundation for robotics applications such as mapping, localization, navigation, and obstacle detection.

---

## ✨ Features

- Real-time LaserScan data acquisition
- YDLIDAR integration with ROS
- Live visualization in RViz
- `/scan` topic subscription
- `base_link` reference frame configuration
- Easy-to-use ROS launch files
- Ubuntu and ROS development environment

## 📂 Project Workflow

1. Connect the YDLIDAR sensor.
2. Launch the YDLIDAR ROS driver.
3. Publish LaserScan data to the `/scan` topic.
4. Open RViz.
5. Visualize real-time LiDAR scans using the LaserScan display.
